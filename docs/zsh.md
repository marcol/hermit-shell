# ZSH

## Aliases

- `pip` - alias for pipx
- `reload` - reload zsh configuration
- `update` - update brew and node

## Oh my shell

[Oh my shell](https://github.com/ohmyzsh/ohmyzsh/wiki/Plugins) is used to manage Plugins.
